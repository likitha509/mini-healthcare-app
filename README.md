# Mini Healthcare Support Web App

## 📌 Description
This is a simple healthcare support web app designed for NGOs to connect patients and volunteers.

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 🤖 AI Feature
A simple rule-based chatbot that answers frequently asked questions like appointments and help.

## ✨ Features
- Patient/Volunteer form
- Auto response after submission
- Chatbot for FAQs
- Clean UI

## 🌍 Use Case
Helps NGOs collect patient requests and provide quick assistance through volunteers.
